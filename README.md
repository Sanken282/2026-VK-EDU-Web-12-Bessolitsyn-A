Как запустить проект:
1.  Откройте папку public в файловом менеджере.
2.  Найдите файл index.html и откройте его - он откроется в браузере.

Страницы (роуты) проекта:
  Главная '/': 'public/index.html'
  Вопросы '/question/': 'public/question.html'
  Задать вопрос '/ask/': 'public/ask.html'
  Вход '/login/': 'public/login.html'
  Регистрация '/signup/': 'public/signup.html'

Структура проекта:

public/
   index.html
   question.html
   ask.html
   login.html
   signup.html
   base.html
   static/
   css/
   js/
   img/
.gitignore
README.md
